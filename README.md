# mastodon-wordcloud
mastodonで1日のtootを取得しWordCloudを作成します。

Mastodonの東方インスタンス[gensokyo.cloud](https://gensokyo.cloud)で動いているbot,幻想トレンド([@genso_trend](https://gensokyo.cloud/@genso_trend))のプログラムです。
